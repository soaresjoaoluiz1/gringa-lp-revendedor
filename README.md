# Gringa — Landing Page Revendedor

Landing page de captação para farmácias, perfumarias, lojas e distribuidoras que querem vender a linha Gringa Cosmético (Body Splash + Creme Hidratante).

## Stack
- HTML5 estático
- CSS inline (mobile-first, breakpoint `@media (min-width: 900px)`)
- SVGs inline (sem dependências)
- Fonts: Inter (texto) + Fraunces (display)
- Imagens em `assets/` (mesmas da LP gringa.ind.br/promo)
